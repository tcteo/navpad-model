A 4x5 nav/macro keyboard.

- Fits MX switches.
- Designed for DSA profile keycaps, but I expect other profiles will work fine too.
- [OpenSCAD](https://openscad.org/) generated in Python, using [scadgen](https://github.com/tcteo/scadgen).
